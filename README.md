# Final Project: Website 

This project is my personal portfolio website, created to showcase my coursework, technical skills, and projects as a Business Analytics & Information Systems student at the University of Iowa.

## Purpose
The website serves as a professional portfolio that highlights my background in analytics, web development, and data-driven problem solving.

## Features
- Multi-page responsive layout
- Clean navigation and structure
- Project descriptions and visuals
- Deployed as a static website

## Technologies Used
- HTML
- CSS
- Bootstrap
- GitHub

## What I Learned
This project helped me improve my front-end development skills, understand website structure and usability, and practice version control using GitHub.
